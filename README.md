# Moe-software

## Order and Voucher Management System

Desktop application developed in Python (PySide6) with a backend built on Django + Django REST Framework and a PostgreSQL database.
It allows managing customer orders, updating existing ones, and automatically generating delivery vouchers in .docx format.

### Main Features

- Order Management: create, update, and delete orders with full backend synchronization.
- Books and Additives Control: dynamic calculation of prices, discounts, and quantities.
- Voucher Generation: visual voucher creation including order, client, and payment details.
- REST API Integration: consumes API endpoints to fetch and update order data.

### Key Technologies

- Frontend: PySide6 (Qt for Python).
- Backend: Django, Django REST Framework, PostgreSQL.
